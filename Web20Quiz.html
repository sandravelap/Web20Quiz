WEB 2.0

¿Qué es la web 2.0?

La Web 2.0 supone la democratización de internet, todos podemos publicar contenido en ella. Su característica principal es la interactividad. Las páginas webs ahora son dinámicas son y nos permiten escribir comentarios, subir fotos, añadir entradas, publicar opiniones, introducir datos en un formulario, etc. Forman parte de la Web 2.0 las herramientas colaborativas (como los servicios Docs o Calendar de Google), los repositorios de imágenes, presentaciones o vídeos (SlideShare), los blogs, las wikis, las redes sociales, etc.

Las Wikis

Una wiki es un sitio web construido de forma colaborativa con el objetivo de crear un lugar de conocimiento. Ejemplo: La Wikipedia que está realizada gracias a la aportación de millones de usuarios de todo el mundo. Cada página editada llevará un historial con los cambios realizados indicando el usuario que los realizó. Si no nos gusta la versión actual de la página siempre es posible volver a una versión anterior. Google Sites nos permite hacer wikis de una manera fácil.

Los Blogs

Los blogs permiten expresar opiniones en un sitio web y compartirlas, debatiéndolas y contrastándolas con los de otros usuarios, pero siempre bajo el control del creador del blog que, en última instancia, es el que decide lo que puede publicarse o no en él. Los blogs se crean a base de entradas que se muestran en orden cronológico inverso. Es útil asignar a cada entrada una categoría según la temática de la que traten. Los sitios de Internet más populares para hacer blogs graturitos son Blogger y Wordpress.

Las redes sociales

Las redes sociales son un fenómeno del siglo XXI. En los últimos años se ha incrementando de forma considerable el uso de las comunicaciones a través de Internet entre los jóvenes, especialmente de las redes sociales.
Hay muchos tipos de redes sociales, aunque las más populares son las que usan para el ocio, como Facebook o Google+. Hay otro tipo de redes sociales más especializadas en temas como linkedin para contactos profesionales, Youtube, Instagram o WhatsApp.

Las páginas web

Las páginas web, también llamadas documentos HTML, son los documentos que muestran los servidores web en nuestros navegadores (Edge, Firefox, Google Chrome, etc).  Los documentos HTML deben de ser creados en un lenguaje específico, el lenguaje HTML (Hyper Text Markup Language),que garantiza una correcta visualización con independencia del navegador utilizado.
Los editores de páginas webs permiten realizar documentos HTML fácilmente y, muchas veces, sin necesidad de conocer el lenguaje HTML.

La computación en la nube

El auge de Internet ha propiciado la posibilidad de conseguir múltiples servicios web 2.0 a través de esta red. A estos servicios se les ha llamado “la nube”. Los servicios pueden ir desde manejar aplicaciones, crear documentos o crear archivos. Ejemplos de estos servicios serían Google Drive, Office 365 o Dropbox. Con Google Drive se tiene una herramienta potente para que el usuario crear documentos. Además se pueden compartir bien para ser vistos o bien para ser editados de forma conjunta.

Plataformas de trabajo colaborativo

Se entiende el trabajo colaborativo como aquel en el que participan varios usuarios, formando un grupo, para realizar una tarea, aunque es más que repartirse el trabajo ya que todos los participantes podrían colaborar para mejorar cualquier aspecto del trabajo aunque haya sido iniciado por otro usuario. Para el trabajo colaborativo se suelen usar las herramientas disponibles en las plataformas virtuales.

Realidad aumentada

La realidad aumentada es el término que se usa para definir una visión a través de un dispositivo tecnológico, directa o indirecta, de un entorno físico del mundo real, cuyos elementos se combinan con elementos virtuales para la creación de una realidad mixta en tiempo real. Los dispositivos añaden información virtual a la información física.

Internet de las cosas (IOT)

Internet de las cosas es un concepto que se refiere a la interconexión digital de objetos cotidianos con internet. Alternativamente, Internet de las cosas es el punto en el tiempo en el que se conectarían a internet más “cosas u objetos” que personas.
